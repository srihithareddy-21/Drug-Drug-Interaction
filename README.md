# AI-Based Drug–Drug Interaction Detection System

## Project Overview
Many patients take multiple medicines prescribed by different doctors or combine prescription drugs with over-the-counter medications without fully understanding possible drug–drug interactions (DDIs). These interactions can lead to serious side effects, reduced effectiveness of medications, or emergency health situations.

Healthcare professionals typically rely on complex medical databases to check drug interactions, which are not easy for general users to understand. This project aims to solve that problem by developing a simple and user-friendly AI system that can detect potential drug interactions and explain the risks in clear, everyday language.

## Objective
The objective of this project is to build an AI-powered system that allows users to input drug names and receive information about possible interactions along with easy-to-understand explanations.

## System Features
- Detect potential drug–drug interactions
- Provide simple explanations of interaction risks
- User-friendly interface for entering drug names
- AI-powered analysis for better interaction detection

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript / React JS

### Backend
- Node.js (for handling requests and storing user data)

### AI Models (Hugging Face)
- Bio_ClinicalBERT
- T5-small

## Workflow
1. User enters the names of two or more drugs.
2. The system processes the input using AI models.
3. The model detects potential drug interactions.
4. The system generates a clear explanation for the user.

## Applications
- Drug safety awareness
- Healthcare assistance tools
- Clinical decision support
- Educational medical tools
